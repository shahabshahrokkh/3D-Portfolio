# نمای بصری چیدمان قفسه 📚

## 🎨 نمای سه‌بعدی

```
نمای از جلو (رو به قفسه):

    ┌─────────────────────────┐
    │                         │
    │  🌐 HTML/CSS/JS         │ ← قفسه بالا (Y: 1.5m)
    │  (چپ)                   │
    ├─────────────────────────┤
    │                         │
    │              ⚛️ React   │ ← قفسه وسط (Y: 1.0m)
    │              (راست)     │
    ├─────────────────────────┤
    │                         │
    │  🐍 Python              │ ← قفسه پایین (Y: 0.5m)
    │  (چپ)                   │
    └─────────────────────────┘
         ↑
    [زمین - Y: 0]
```

## 🎯 مختصات دقیق

### قفسه کتاب:
```

```

#### 🐍 Python (قفسه پایین - چپ)
```

Z: -7.15 (5cm جلوتر از دیوار - داخل قفسه)
```

#### ⚛️ React (قفسه وسط - راست)
```

Z: -7.15 (5cm جلوتر از دیوار - داخل قفسه)
```

#### 🌐 HTML/CSS/JS (قفسه بالا - چپ)
```
X: -3.1  (10cm سمت چپ از مرکز قفسه)
Y:  1.5  (150cm از زمین - قفسه بالا)

```

## 💡 نکات مهم

### ⚠️ اگر آیکون‌ها دیده نمی‌شوند:
1. Z خیلی عقب است → داخل دیوار
2. Y خیلی پایین است → زیر قفسه
3. اندازه خیلی کوچک است

### ⚠️ اگر آیکون‌ها بیرون از قفسه هستند:
1. Z خیلی جلو است → از -7.15 به -7.16 تغییر دهید
2. X خیلی دور است → به -3.0 نزدیک‌تر کنید

### ⚠️ اگر آیکون‌ها با هم تداخل دارند:
1. Y ها یکسان هستند → ارتفاع‌های مختلف استفاده کنید
2. X ها خیلی نزدیک هستند → فاصله بیشتر دهید
