# 🎨 افزایش فاصله Border از محتوا

## ❌ مشکل
border رنگی (بنفش، آبی، ...) که بعد از کلیک روی پروژه‌ها نمایش داده می‌شود، خیلی نزدیک به متن بود.

---

## ✅ راه‌حل
افزایش `padding` آیتم‌ها برای ایجاد فاصله بیشتر بین border و محتوا.

---

## 🔧 تغییرات

### 1️⃣ Base Style (Desktop)
```css
/* قبل ❌ */
.po-list-item {
  padding: 12px 14px;  /* خیلی نزدیک */
}

/* بعد ✅ */
.po-list-item {
  padding: 16px 18px;  /* فاصله بیشتر */
}
```

**افزایش**: +33% عمودی، +29% افقی

### 2️⃣ Tablet (< 768px)
```css
/* قبل ❌ */
.po-list-item {
  padding: 18px 20px;
}

/* بعد ✅ */
.po-list-item {
  padding: 20px 22px;  /* فاصله بیشتر */
}
```

**افزایش**: +11% عمودی، +10% افقی

### 3️⃣ موبایل (< 480px)
```css
/* قبل ❌ */
.po-list-item {
  padding: 20px 18px;
}

/* بعد ✅ */
.po-list-item {
  padding: 22px 20px;  /* فاصله بیشتر */
}
```

**افزایش**: +10% عمودی، +11% افقی

### 4️⃣ موبایل کوچک (< 375px)
```css
/* قبل ❌ */
.po-list-item {
  padding: 18px 16px;
}

/* بعد ✅ */
.po-list-item {
  padding: 20px 18px;  /* فاصله بیشتر */
}
```

**افزایش**: +11% عمودی، +13% افقی

## 💡 چرا این مهم است؟

### 1. خوانایی بهتر
- متن از border فاصله دارد
- چشم راحت‌تر می‌تواند بخواند
- کمتر گیج‌کننده

### 2. زیبایی بصری
- تعادل بهتر
- فضای تنفس (breathing room)
- طراحی حرفه‌ای‌تر

### 3. تجربه کاربری
- راحت‌تر قابل کلیک
- مشخص‌تر که کدام آیتم انتخاب شده
- border به عنوان frame عمل می‌کند نه محدودیت

## 🎯 قانون طراحی

### فاصله مناسب Border
```
حداقل padding برای border واضح:
- Desktop: 16px (عمودی)
- Mobile: 20px (عمودی)

قانون کلی:
padding ≥ 1.5 × border-width × 10
```

### مثال
```css
.po-list-item {
  border: 1px solid var(--accent);
  padding: 16px 18px;  /* 16 ≥ 1.5 × 1 × 10 ✅ */
}
```
