## مشخصات فنی:

```javascript
// نوع هندسه
IcosahedronGeometry(radius, subdivisions)

// پارامترها
radius = √((11.5/2)² + (11.5/2)²) ≈ 8.13
subdivisions = 2  // سطح جزئیات

// نتیجه
- تعداد مثلث‌ها: ~80 (برای نیم‌کره)
- تعداد خطوط: ~120
- شکل: ساختار ژئودزیک با مثلث‌های متساوی‌الاضلاع
```

## ویژگی‌های ساختار ژئودزیک:

### 1. مثلث‌های متصل
- هر راس به چند راس دیگر متصل است
- خطوط مورب و افقی و عمودی
- ساختار قوی و یکپارچه

### 2. توزیع یکنواخت
- مثلث‌ها تقریباً هم‌اندازه هستند
- فاصله یکنواخت بین راس‌ها
- ظاهر حرفه‌ای و معماری

### 3. نیم‌کره کامل
- فقط قسمت بالای زمین نمایش داده می‌شود
- راس‌های زیر زمین به سطح کف clamp می‌شوند
- اتصال یکپارچه به کف
