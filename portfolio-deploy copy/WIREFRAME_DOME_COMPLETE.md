# گنبد و کف دایره‌ای وایرفریم - تکمیل شده ✓

## وضعیت: کامل شد

### تغییرات اعمال شده:

#### 1. کف دایره‌ای (Circular Floor)
- ✅ کف از مربع به دایره تبدیل شد
- ✅ شامل 8 دایره متحدالمرکز (concentric circles)
- ✅ شامل 24 خط شعاعی (radial lines) از مرکز به لبه
- ✅ شعاع دقیقاً با قطر گنبد برابر است
- ✅ فرمول شعاع: `√((width/2)² + (depth/2)²)` = `√((11.5/2)² + (11.5/2)²)` ≈ 8.13

#### 2. گنبد ژئودزیک (Geodesic Dome)
- ✅ 8 حلقه افقی
- ✅ 24 خط عمودی
- ✅ شعاع دقیقاً با کف دایره‌ای برابر
- ✅ لبه گنبد دقیقاً روی لبه کف قرار می‌گیرد

#### 3. دیوارها (Walls)
- ✅ دیوار عقب (back wall) با الگوی شبکه‌ای
- ✅ دیوار چپ (left wall) با الگوی شبکه‌ای
- ✅ 15×15 تقسیم‌بندی برای هر دیوار

### مشخصات فنی:

```javascript
// ابعاد اتاق
ROOM_CONFIG = {
    width: 11.5,      // محور X
    depth: 11.5,      // محور Z
    height: 5,        // محور Y
    centerX: -1.75,   // مرکز X
    centerZ: -1.75    // مرکز Z
}

// محاسبه شعاع
radius = √((11.5/2)² + (11.5/2)²) ≈ 8.13

// کف دایره‌ای
- حلقه‌ها: 8
- خطوط شعاعی: 24
- سگمنت‌های دایره: 48 (برای نرمی)

// گنبد
- حلقه‌های افقی: 8
- خطوط عمودی: 24
- نوع: نیم‌کره ژئودزیک
```

### ظاهر بصری:

- **رنگ**: سفید (#ffffff)
- **شفافیت**: 0.12 - 0.15
- **انیمیشن**: پالس نرم (opacity oscillation)
- **چرخش**: گنبد به آرامی می‌چرخد (0.0002 rad/frame)

### فایل‌های تغییر یافته:

1. `src/scene/wireframeDome.js` - پیاده‌سازی کامل
2. `src/main.js` - یکپارچه‌سازی و انیمیشن

### نتیجه:

کف اتاق حالا **دقیقاً دایره‌ای** است و **اندازه آن با لبه گنبد برابر** است. هر دو از یک فرمول شعاع استفاده می‌کنند و کاملاً هم‌اندازه هستند.
