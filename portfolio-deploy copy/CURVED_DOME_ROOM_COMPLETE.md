## مشخصات فنی:

```javascript
// ابعاد و محاسبات
ROOM_CONFIG = {
    width: 11.5,
    depth: 11.5,
    height: 5,
    centerX: -1.75,
    centerZ: -1.75
}

// شعاع یکسان برای همه اجزا
radius = √((11.5/2)² + (11.5/2)²) ≈ 8.13

// کف دایره‌ای
CircleGeometry(radius, 48)

// دیوارهای قوسی
- نوع: Custom BufferGeometry
- شکل: نیم‌دایره عمودی (quarter-sphere arc)
- سگمنت‌های افقی: 24
- سگمنت‌های عمودی: 16
- زاویه عمودی: 0° تا 90° (کف تا بالای گنبد)
- زاویه افقی: -90° تا +90° (نیم‌دایره)

// وایرفریم دیوارها
- خطوط افقی: 8
- خطوط عمودی: 24
- شکل: قوس‌های منحنی
```
