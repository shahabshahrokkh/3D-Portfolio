# خلاصه: اضافه شدن قفسه کتاب 📚

## ✅ انجام شد

یک قفسه کتاب جدید (`room_shelves.glb`) به صحنه 3D اضافه شد.

---

## 📍 موقعیت

```
        [دیوار عقب]
    
    وایتبورد        قفسه کتاب        آرکید
       (0)            (-3)           (-6)
        |              |              |
        |              📚             |
        |                             |
    [میز کامپیوتر]              [گوشه]
```

**مختصات دقیق:**
- X: -3 (بین میز و آرکید)
- Y: 0 (روی زمین)
- Z: -7.2 (چسبیده به دیوار عقب)

---

## 📝 فایل‌های تغییر یافته

### 1. ✅ `src/utils/config.js`
اضافه شدن تنظیمات مدل bookshelf

### 2. ✅ `src/objects/bookshelf.js` (جدید)
ماژول بارگذاری قفسه کتاب

### 3. ✅ `src/main.js`
Import و فراخوانی initBookshelf()

---

## 🎨 مشخصات

- **ارتفاع:** 2 متر
- **نوع:** محیطی (غیرقابل تعامل)
- **چرخش:** 180° (رو به داخل اتاق)
- **سایه:** فعال

---

## 🧪 تست

برای تست، پروژه را اجرا کنید:

```bash
npm run dev
```

قفسه کتاب باید:
- ✅ بین میز و آرکید ظاهر شود
- ✅ به دیوار عقب چسبیده باشد
- ✅ ارتفاع مناسب داشته باشد
- ✅ سایه بیندازد
