### 1. دکمه Toggle با آیکون همبرگر (3 خط)


- تبدیل همبرگر به X
- باز/بسته شدن پنل با حرکت نرم
- استفاده از cubic-bezier برای حرکت طبیعی

### 3. ذخیره وضعیت

- یادآوری ترجیح کاربر در بازدیدهای بعدی

### 4. بازخورد لمسی
- ویبره 30ms هنگام باز/بسته کردن پنل

## 🧪 تست

### چگونه تست کنیم؟

1. **اجرای پروژه**:
```bash
npm run dev
```

2. **باز کردن در موبایل**:
   - Chrome DevTools → Device Mode
   - یا اسکن QR code برای تست روی دستگاه واقعی

3. **تست عملکرد**:
   - ✅ کلیک روی دکمه toggle
   - ✅ بررسی انیمیشن همبرگر → X
   - ✅ بررسی باز/بسته شدن نرم پنل
   - ✅ بررسی ویبره در موبایل
   - ✅ رفرش صفحه و بررسی حفظ وضعیت
