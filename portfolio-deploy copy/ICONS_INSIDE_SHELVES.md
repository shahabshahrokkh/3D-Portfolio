# آیکون‌ها داخل قفسه‌ها 📚💻

## ✅ اصلاح شد

آیکون‌ها حالا **داخل** قفسه‌ها قرار دارند، هر کدام در قفسه‌ای جداگانه با ارتفاع متفاوت.

---

## 📍 چیدمان جدید

```
نمای از جلو:

    ┌─────────────────┐
    │  🌐             │ ← قفسه بالا (1.5m) - HTML/CSS/JS چپ
    ├─────────────────┤
    │           ⚛️    │ ← قفسه وسط (1.0m) - React راست
    ├─────────────────┤
    │  🐍             │ ← قفسه پایین (0.5m) - Python چپ
    └─────────────────┘
```

## 📐 موقعیت‌های جدید

### 🐍 Python Icon
```javascript
position: [-3.2, 0.5, -7.15]
```
- **X: -3.2** → سمت چپ قفسه
- **Y: 0.5** → قفسه پایین
- **Z: -7.15** → داخل قفسه (5 سانتی‌متر جلوتر از دیوار)

- **Z: -7.15** → داخل قفسه

- **X: -3.1** → سمت چپ قفسه

- **Z: -7.15** → داخل قفسه

## 🎯 منطق چیدمان

### چرا این موقعیت‌ها؟

**1. عمق (Z = -7.15):**
- قفسه در Z = -7.2
- آیکون‌ها 5 سانتی‌متر جلوتر → داخل قفسه
- نه خیلی عقب (داخل دیوار)
- نه خیلی جلو (بیرون از قفسه)

**2. ارتفاع‌های متفاوت:**
- قفسه پایین: 0.5m
- قفسه وسط: 1.0m
- قفسه بالا: 1.5m
- فاصله یکسان: 0.5m بین هر قفسه

**3. موقعیت افقی متنوع:**
- Python: چپ (-3.2)
- React: راست (-2.8)
- HTML: چپ (-3.1)
- ایجاد تنوع بصری

**4. اندازه کوچک‌تر:**
- از 0.25m به 0.2m
- بهتر در قفسه جا می‌شوند
- واقعی‌تر به نظر می‌رسند

## 🎨 تنظیمات پیشرفته

### تغییر عمق (جلو/عقب در قفسه):
```javascript
// جلوتر (نزدیک به لبه قفسه):
position: [-3.2, 0.5, -7.1]

// عقب‌تر (نزدیک به دیوار):
position: [-3.2, 0.5, -7.18]
```

### تغییر ارتفاع قفسه‌ها:
```javascript
// قفسه‌های نزدیک‌تر به هم:
pythonIcon:  Y = 0.4
reactIcon:   Y = 0.9
htmlIcon:    Y = 1.4

// قفسه‌های دورتر از هم:
pythonIcon:  Y = 0.5
reactIcon:   Y = 1.1
htmlIcon:    Y = 1.7
```

### تغییر موقعیت افقی:
```javascript
// همه در وسط:
X = -3.0

// همه در چپ:
X = -3.2

// همه در راست:
X = -2.8
```

## 💡 ایده‌های بهبود

### 1. چرخش متفاوت:
```javascript
// Python: کمی به چپ
rotation: [0, Math.PI + 0.2, 0]

// React: کمی به راست
rotation: [0, Math.PI - 0.2, 0]
```

### 2. اندازه‌های متفاوت:
```javascript
pythonIcon:  { height: 0.18 }  // کوچک‌تر
reactIcon:   { height: 0.22 }  // بزرگ‌تر
htmlIcon:    { height: 0.20 }  // متوسط
```

### 3. افکت نور:
```javascript
// اضافه کردن SpotLight برای هر آیکون
const pythonLight = new THREE.SpotLight(0x3776ab, 0.3);
pythonLight.position.set(-3.2, 0.7, -7.0);
pythonLight.target.position.set(-3.2, 0.5, -7.15);
```
